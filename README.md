# ClassicHCAS
## The Classic Habitat Condition Assessment System (HCAS)

The package `ClassicHCAS` offers a range of functions for generating habitat condition...

**Note: the package is still under development, so the function might not work and the user interface might completely change.**

## Installation

The `Rtools` toolchain is required to build the package on Windows, see [the following link](https://cran.r-project.org/bin/windows/Rtools/) for downloading and installing Rtools.

To install the package on a CSIRO system use:
```r
remotes::install_git("https://bitbucket.csiro.au/scm/hcas3/classichcas.git")

```
This might ask your Bitbcket username and password.
